# question5

Fizz Buzz

## 概要

整数 N を入力として
<br>1からNまでの整数を1から順に表示
<br>また、
- 3の倍数かつ5の倍数のときには、"Fizz Buzz"
- 3の倍数のときには、"Fizz"
- 5の倍数のときには、"Buzz"
<br>を数値の代わりに表示
