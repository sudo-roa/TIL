# 変数
- 繰り返し使う値を格納しておくための箱のようなもの
- 変数に値を定義することを、変数を宣言するという

## JavaScriptにおける変数の宣言の仕方
- let

```
let a = 0;
```

- const
 
```
const a = 0;
```
- var

```
var a = 0;
```

## 宣言の仕方ごとの特徴

### let
- 再宣言できない
- 再代入できる
- 値を代入しなければエラー

### const
- 再宣言できない
- 再代入できない
- 値を代入しなければエラー

### var
- 再宣言できる
- 再代入できる
- 値を代入しなければundefinedが格納される
- ES6では非推奨な宣言方法

## 宣言の仕方の違い(表)

| 宣言の仕方 | 再宣言 | 再代入 | 初期化しない場合 |
|:---:|:---:|:---:|:---:|
| let | できない | できる | エラー |
| const | できない | できない | エラー |
| var | できる | できる | undefined |
