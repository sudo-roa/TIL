# PHPにおける演算

PHPにおける足し算、引き算、掛け算、割り算、剰余算、累乗について

## 計算結果の表示


計算結果5が出力される
```
print(2+3);
```
コーテーションを付けると文字列として出力される
```
print("2+3");
```
数学で習ったとおりの計算規則に従って計算が行われる
```
print(1+2*3);
print(1*2+3);
print((1+2)*3);
```

## 演算の種類
| 記号 | 意味  | 例   | 出力結果 |
| -- | --- | --- | ---- |
| +  | 足し算 | 9+2 | 11   |
| -  | 引き算 | 9-2 | 7    |
| *  | 掛け算 | 9*2 | 18   |
| /  | 割り算 | 9/2 | 4.5  |
| %  | 剰余  | 9%2 | 1    |
| ** | 累乗  | 9**2 | 81    |
