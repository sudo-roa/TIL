# 繰り返し構文

PHPにおける繰り返し構文

## for()
### for文の基本的な使い方
```
for(初期化処理; 繰り返す条件; 更新処理){
  繰り返したい処理
}

or

for(初期化処理; 繰り返す条件; 更新処理):
  繰り返したい処理
endfor;
```


## while()
### while文の基本的な使い方
```
初期化処理
while(繰り返す条件){
  繰り返したい処理
  更新処理
}

or

初期化処理
while(繰り返す条件):
  繰り返したい処理
  更新処理
endwhile;
```
