# 再代入

PHPにおける再代入

## 再代入とは

変数にもう一度値を代入しなおすこと

## 四則計算における再代入

### 足し算

```
$i = $i + 1;
$i += 1;
$i ++; //インクリメント
```

### 引き算

```
$i = $i - 1;
$i -= 1;
$i --; //デクリメント
```

### 掛け算

```
$i = $i * 1;
$i *= 1;
```

### 割り算

```
$i = $i / 1;
$i /= 1;
```
